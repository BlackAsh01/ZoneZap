# Guardian live map (OpenStreetMap)

The **Track on map** feature uses **OpenStreetMap** via the OSMDroid library. It is **free** and **no API key** is required.

- **Guardian:** Tap a ward → choose **Track on map** (or tap an alert with location → **Track on map**).
- Map screen: **Refresh** for latest location; **Live** is on by default and auto-refreshes every 5 seconds.

No setup needed—just build and run the app.
